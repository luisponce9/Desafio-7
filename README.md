#Desafio 7

Este proyecto es un desafío práctico correspondiente al curso de Desarrollo Web Frontend del PEAM (UNSE). <br><br>
El objetivo del trabajo es maquetar la interfaz de un sistema de mensajería web aplicando los conocimientos adquiridos en clase. <br><br>
Para su construcción se utilizaron tecnologías como HTML, CSS y Bootstrap, haciendo especial énfasis en el uso de Flexbox como herramienta principal para la estructura, alineación y distribución adaptable de los elementos en pantalla.
